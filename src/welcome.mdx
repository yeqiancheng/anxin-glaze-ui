# 欢迎来到 Cream Design 组件库

Cream Design 是一套现代化的 React 组件库，提供了丰富的 UI 组件和交互体验。从零到一，让您轻松构建高质量的 Web 应用。

## 安装试试

```bash
npm install anxin-glaze-ui --save
```

## 快速开始

```jsx
import { Button, Input, Menu } from "anxin-glaze-ui";
import "anxin-glaze-ui/dist/index.css";

function App() {
  return (
    <div>
      <Button type="primary">点击我</Button>
      <Input placeholder="请输入内容" />
      <Menu>
        <MenuItem>菜单项</MenuItem>
      </Menu>
    </div>
  );
}
```

## 特性

- 🎨 现代化设计，美观易用
- ⚡ TypeScript 支持，类型安全
- 📱 响应式设计，移动端友好
- 🎯 丰富的组件，满足各种需求
- 🔧 高度可定制，灵活配置
